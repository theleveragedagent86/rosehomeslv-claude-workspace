# Responder Comments Daily Schedule -- Paste into Cowork Scheduler

Use this as the prompt when setting up a scheduled task in Cowork for Brian Esposito's daily first-responder community engagement.

**Important:** If you also run `lender-comments` daily, schedule this skill at a different time (e.g., lender at 9 AM, responder at 2 PM) so combined Instagram activity is spread out.

---

## Schedule Settings

- **Frequency:** Daily (or 3-4x per week if combined volume with lender-comments is too high)
- **Time:** Pick a time that works (afternoon recommended if lender-comments runs in the morning)

---

## Prompt to Paste

```
Run the daily first responder community presence routine.

STEP 1 -- OPEN INSTAGRAM
Open Chrome and navigate to https://www.instagram.com. Verify you are logged in as @espohomeloans. If not logged in, stop and let me know.

STEP 2 -- READ TARGET ACCOUNTS
Read the target account list from the first-responder-engagement plugin. Pick 3-5 accounts for today, mixing at least 2 different responder categories (military, police, fire/EMS, healthcare). Rotate so you don't hit the same accounts every day.

STEP 3 -- READ COMMENT GUIDELINES
Read the comment guidelines from the first-responder-engagement plugin. Pay close attention to the rule: NEVER mention mortgages, lending, Nations Lending, VA loans, hero programs, or any business. This is community presence, not lead generation.

STEP 4 -- ENGAGE ON 10-25 POSTS (LIGHTER IF COMBINED WITH lender-comments)
For each target account, open their profile and scroll through recent posts. For each post:
- Screen the content. SKIP ENTIRELY if it touches anything political, controversial, line-of-duty death, active critical incident, venting, religious/divisive, or anything where a comment from a mortgage business account could be misread. Be liberal with skipping.
- If safe: Like the post. Like 3-5 of the most aligned top comments (do not auto-like all). Write and post a 1-2 sentence comment (hard cap at 2, often 1 is better). Voice = warm 28-year Vegas neighbor, not service provider. NEVER mention mortgages or business. Match the post's tone (somber for memorials, warm for milestones, casual for routine).
- Default: do NOT reply to top comments. Only reply if you have something genuinely additive and the tone is clearly safe. Maximum 1 reply per post.
- Wait 5-15 seconds between actions. Wait 10-20 seconds between posts. Wait 30-60 seconds between accounts.
- If Instagram shows "Try Again Later" or any rate limit warning, stop the session immediately and save the log.
- Target 10-25 posts. If lender-comments already ran today, target 5-10 instead.

STEP 5 -- SAVE LOG AND REPORT
Save a session log to output/responder-comments/YYYY-MM-DD.md with: accounts visited, posts engaged, posts skipped with reasons, every comment posted, every reply posted.
Print a summary: accounts visited, posts engaged, posts skipped, comments posted, replies posted, likes given.
```
