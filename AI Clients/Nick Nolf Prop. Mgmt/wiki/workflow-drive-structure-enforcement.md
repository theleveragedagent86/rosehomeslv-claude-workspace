# Workflow: Google Drive Structure Enforcement

## What It Does
Given a perfect example of Nick's folder hierarchy and naming conventions, replicate that structure for every new property, tenant, or investor file going forward.

## Approach
- Nick provides one perfect example folder structure
- Claude replicates it exactly — no guessing, no analyzing existing folders
- Ryan suggested this approach over having Claude analyze 10,000+ folders to infer the pattern
- Deterministic: investor name, address, subfolders, done

## Complexity Assessment
**Tier 1: Straightforward** — This is essentially a template. Define the hierarchy, define the naming convention, replicate exactly. Almost no decision-making involved.
