Based on the workflow established in our sessions, here's the step-by-step process I've been following:
Blog Publishing Workflow:

Add New → Click "+ Add New" button
Title → Enter the blog post title (from the h1 in the HTML file)
Source code HTML → Click the </> icon to open source code editor, then enter HTML content:

h2 "Explore More from Rose Homes LV" with related links at TOP
h2 "Explore More Henderson Communities"
hr separator
Main article content (excluding h1 title and JSON-LD script)


Save → Click Save button in source code modal
Settings tab → Set:

URL slug: lowercase with hyphens, if you try to click publish and nothing happens then the issue is the URL slug you have chosen, already exists and for this case you need to append a "1" at the end of the slug (if this does not fix the issue of publishing, then instead of "1" add a "2" and so on)
Category: Select appropriate category (e.g., "Madeira Canyon")


SEO tab → Set:

Meta Title: Remove default variables by doing ctrl+a, enter custom title from file
Meta Keywords: Enter keywords from file
Meta Description: Remove "Blog First Sentence" variable by doing ctrl+a, enter custom description


Post Now → Click to publish
OK → Confirm if dialog appears

Key Notes:

Ctrl+A to select all text in fields before replacing
