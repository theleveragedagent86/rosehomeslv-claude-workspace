# Coffee & Contracts Monthly Edit — Default Slide Map

This file defines the standard slide types in a C&C "Monthly Edit" Canva multi-design. Slide numbers may shift month to month as C&C updates the template. The user confirms or overrides these numbers at the start of each run.

## Standard Slide Assignments (May 2026 baseline)

| Slide # | Type | Has AI Prompt? | Research Source |
|---------|------|----------------|-----------------|
| 13 | Cover | No | Topical image suggestion from Events research |
| 16 | Events | Yes | Events Research Agent |
| 17 | New & Coming Soon | Yes | Local Research Agent |
| 18 | Bucket List | Yes | Local Research Agent |
| 21 | Housing Market | Yes | Market Stats Research Agent |
| 22 | CTA / Closing | No | Standard template (no research needed) |

## Slide Type Definitions

### Cover
- Bold title overlay: `[MONTH] IN LAS VEGAS GUIDE`
- Background photo: topical image matching the month's biggest event or season
- No AI prompt, but the Events research suggests a cover image theme

### Events
- Header: `[Month] Events in Las Vegas`
- 10 events split into 2 columns (5 left, 5 right)
- Each event: name (bold), date(s) and time, venue/location
- Footer CTA: `COMMENT "LAS VEGAS" FOR THE FULL SPRING GUIDE`

### New & Coming Soon
- Header: `New & Coming Soon in Las Vegas`
- 5 bullet items, each with name (bold) and 1-sentence description
- Focus: restaurant openings, new businesses, notable debuts

### Bucket List
- Header: `[Month] Bucket List in Las Vegas`
- 4 categories: Kids, Adults, To Eat, To Drink
- Each category: 2-3 sentences with specific venue, detail, and seasonal relevance

### Housing Market
- Header: `The scoop on the housing market`
- 3 stat boxes: Median Sold Price, Avg Days on Market, List-to-Sale Price Ratio
- Each box: value, label, trend arrow with context
- Bottom narrative: "so what's that mean for you?" with buyer and seller paragraphs
- All numbers must have source citations below

### CTA / Closing
- Header: `How can I help you this month?`
- Soft closing message
- Ryan's contact info
- No research needed, uses standard template

## Notes

- The Canva template uses `[City Name]` as a placeholder. Bulk replace with `Las Vegas` via Cmd+F before pasting content.
- The template uses the previous month's name. Bulk replace with the target month via Cmd+F.
- Yellow "Head to the AI prompt" sticky notes appear on slides that need researched content. Delete these before posting.
