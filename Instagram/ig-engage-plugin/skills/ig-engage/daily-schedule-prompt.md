# Instagram Engagement Daily Schedule -- Paste into Cowork Scheduler

Use this as the prompt when setting up a scheduled task in Cowork.

---

## Schedule Settings

- **Frequency:** Daily
- **Time:** Pick a time that works (morning recommended, e.g., 9:30 AM)

---

## Prompt to Paste

```
Run the daily Instagram engagement routine.

STEP 1 -- OPEN INSTAGRAM
Open Chrome and navigate to https://www.instagram.com. Verify you are logged in. If not logged in, stop and let me know.

STEP 2 -- READ TARGET ACCOUNTS
Read the target account list at /Users/ryanrose/.claude/skills/ig-engage/target-accounts.md. Pick 3-5 accounts for today. Rotate so you don't hit the same accounts every day.

STEP 3 -- READ COMMENT GUIDELINES
Read /Users/ryanrose/.claude/skills/ig-engage/comment-guidelines.md for writing guidance and tone examples.

STEP 4 -- ENGAGE ON 10-25 POSTS
For each target account, open their profile and scroll through recent posts. For each post:
- Screen the content. SKIP ENTIRELY if it touches anything political, LGBTQ+, or highly controversial. Do not like, do not comment, just move on.
- If safe: Like the post. Like the top 5-10 comments. Write and post a value-add comment (1-3 sentences, adaptive tone, genuine, not salesy, no em-dashes ever). Reply to 1-2 top comments if there is something genuine to add.
- Wait 5-15 seconds between actions.
- If Instagram shows "Try Again Later" or any rate limit warning, stop the session immediately.
- Target 10-25 posts total across all accounts.

STEP 5 -- REPORT
Output a summary: accounts visited, posts engaged, posts skipped (guardrails), comments posted, replies posted, likes given.
```
