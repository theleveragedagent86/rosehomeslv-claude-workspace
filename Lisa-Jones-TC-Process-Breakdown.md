## KEY PATTERNS FOR AUTOMATION

### Cadence Rules

- Lisa never lets more than **1 business day** pass without following up on open items
- Her follow-up cycle is aggressive: request on Friday, follow up Monday if no response
- Multiple parallel workstreams run simultaneously (inspection scheduling, escrow coordination, commission QA)

### Reusable Templates (Prime Automation Candidates)

1. **Accepted Offer Email to Buyer** - sent immediately on deal intake
2. **Home Inspection Options Email** - sent same day as accepted offer
3. **Introduction to All Transaction Parties** - sent once escrow officer info is obtained
4. **Preliminary Title Report Email to Buyer** - sent once prelim is received from title
5. **Inspection Report Follow-up to Agent** - sent morning after report is received

### Trigger Map

| Trigger Event | Action |
|---|---|
| Agent submits deal docs to deals@ | Kicks off entire Phase 1 (7 emails in ~2 hours) |
| Buyer selects inspector | Initiates inspection coordination email to builder + inspector |
| Escrow officer info received | Sends intro email to all transaction parties |
| Opening package received from title | QA's commission documents for accuracy |
| Prelim title report received | Forwards to buyer with template explanation |
| Inspection report received | Asks agent about next steps / repairs |
| Deal status uncertain | Check in within 24 hours |
| Cancellation confirmed | Instructs agent on file cleanup (cancel in Rezen, send cancellation docs) |

### Communication Lanes

| Recipient | Type of Communication |
|---|---|
| **Agent (Ryan)** | Status updates, commission questions, action items, deal status checks |
| **Buyer (Noah)** | Templated milestone emails with attachments (accepted offer, prelim title, inspection options) |
| **Builder Rep (Raquel)** | Escrow/lender info requests, inspection coordination, W9, Request for Entry forms |
| **Escrow Officer (Denise)** | Commission instructions, opening packages, TC invoice |
| **Inspector (Nikki)** | Scheduling, Request for Entry forms, access coordination, buyer contact info |

### Systems Used

- **Rezen** - deal entry, document uploads, transaction management
- **Authentisign** - electronic signatures (Request for Entry forms)
- **DocuSign** - builder-side contract signatures
- **Microsoft Teams** - calendar invites for COE
- **Email (deals@altgrouplv.com)** - team distribution list for all transaction communication
