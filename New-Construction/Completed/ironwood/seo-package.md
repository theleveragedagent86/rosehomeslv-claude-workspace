## 4. Per-Plan Blog Guidance (passed to Blog Writer)

### Title pattern
`[Plan Name] at Ironwood | [Sqft] sq ft from $[Price] in SW Las Vegas`

Examples (54-60 chars):
- `Residence 1684 at Ironwood | 1,684 sq ft from $495,990 SW Las Vegas`
- `Residence 1859 at Ironwood | 1,859 sq ft New Home in 89113 Las Vegas`
- `Residence 2001 at Ironwood | 2,001 sq ft Century Communities Las Vegas`
- `Residence 2114 at Ironwood | 2,114 sq ft up to 6 BR in 89113 Las Vegas`

### Description pattern
`The [Plan Name] at Ironwood by Century Communities is a [Sqft] sqft, [Beds]-bed, [Baths]-bath two-story home in 89113 Southwest Las Vegas, priced from $[Price]. Up to $50K in incentives via Inspire Home Loans. Tour with Ryan Rose, Real Broker LLC.`

(Compress to 150-155 chars; trim "Real Broker LLC" if over.)

### Per-plan keyword formulas

For each plan use:
- **Brand + plan:** `Residence [N] Ironwood`, `Century Communities Residence [N]`
- **Plan + city:** `Residence [N] Las Vegas`, `Residence [N] 89113`
- **Plan + sqft + beds:** `[Sqft] sqft new home Las Vegas`, `[Beds] bedroom new construction 89113`
- **Plan + price:** `new home under $[Price ceiling] Las Vegas`
- **Plan + area:** `Residence [N] Spring Valley`, `Residence [N] near Mountain's Edge`
- **Question:** `Is the Residence [N] worth it`, `What does the Residence [N] include`
- **Comparison:** `Residence [N] vs Residence [N+1]`, `Ironwood vs Cambria Bay`

### Blog Writer instruction

For each of the 4 plan blogs, Blog Writer should:

1. Set `<title>` and `<meta description>` using the patterns above with the plan-specific facts from `research-report.md`.
2. Use Article schema (Blog Writer plugin's default) and add:
   - `"about": { "@type": "Product", "name": "[Plan Name] at Ironwood", "brand": "Century Communities", "offers": { "price": "[Price]", "priceCurrency": "USD" } }`
   - `"author": { "@type": "Person", "name": "Ryan Rose", "telephone": "+1-702-747-5921", "email": "ryan@rosehomeslv.com" }`
   - `"publisher": { "@type": "RealEstateAgent", "name": "Real Broker, LLC" }`
3. Internal-link back to `https://rosehomeslv.com/century-ironwood` (community LP) and `https://rosehomeslv.com/new-construction-las-vegas` (master).
4. Include 2 contextual internal links from the Reference Data list (see "Internal Linking" below).
5. Include H2s shaped like questions (AEO): "How big is the Residence [N]?", "What's included at this price?", "How does it compare to other Ironwood plans?", "What schools serve Ironwood?".
6. End with a soft CTA: "Want to walk this plan in person? Text Ryan at 702-747-5921 or email ryan@rosehomeslv.com."
7. No em-dashes. No "premier" / "luxury" / "exclusive". 6th-grade reading level.
