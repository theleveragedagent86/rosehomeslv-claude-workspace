## Lofty CRM Setup Instructions

### Step 1: Create the Action Plans
Create 3 separate action plans in Lofty CRM:
1. `Ironwood - Hot`
2. `Ironwood - Warm`
3. `Ironwood - Nurture`

### Step 2: Set the Trigger
For each plan, set the trigger to: **Tag Applied = `ironwood`** (lowercase, exact spelling).
Track assignment is based on the lead's qualifying answers from the Meta Instant Form (timeline + pre-approval). For landing page leads with no qualifying answers, assign to the Warm track by default.

### Step 3: Add Actions
Add each touch from the sequence above as an action in the plan:
- **Text Message actions:** Set the delay, paste the copy exactly, enable business hours (9:00 AM to 7:00 PM PT).
- **Email actions to the lead:** Set the delay, paste the subject and body, set sender as Ryan Rose (ryan@rosehomeslv.com).
- **Call Task and Voicemail Task actions:** Set the delay and time, paste the task description for Ryan.
- **Day 0 email to the lead (Touch 2):** Attach the `Duties Owed and BBRA.pdf` from `/Users/ryanrose/Downloads/Claude/New-Construction/DO.BBRA - Template/Duties Owed and BBRA.pdf`.
- **Day 0 builder rep notification (Touch 3):** Set the recipient to ryan@rosehomeslv.com, NOT the lead.
- **Day 35 handoff (Touch 17):** Use the "Start Another Action Plan" action type and select `Long Term Nurture - Buyer`.

### Step 4: Configure Settings
- Enable **Stop on Reply** for all 3 plans.
- Set business hours for text messages: 9:00 AM to 7:00 PM PT.
- Email sender: Ryan Rose (ryan@rosehomeslv.com).
- Activate (publish) each plan after building.

### Step 5: Test
- Create a test contact in Lofty.
- Apply the tag `ironwood` to the test contact.
- Verify the action plan triggers and the Touch 1 text and Touch 2 email fire correctly.
- Verify the Touch 3 builder rep notification email arrives at ryan@rosehomeslv.com.
- Verify the `Duties Owed and BBRA.pdf` is attached to the Day 0 lead email (Touch 2).
- Verify the Day 35 handoff is wired to `Long Term Nurture - Buyer`.
- Delete the test contact when done.
