## How to use this file

1. Before running `/new-construction [Community Name]`, scan the table below.
2. If the community has a ✅ in the Status column, it is DONE. Skip it.
3. If it has 🔄 (in progress) or is missing from the table, it has not been completed.
4. After finishing a community, update its row: change status to ✅, fill in the output folder name and completion date.
5. Output folder lives at `/Users/ryanrose/Downloads/Claude/New-Construction/[output-folder]/`.
