# Module 0: Start Here — The Leveraged Agent

## Module Overview

This is the onboarding module every new member hits first. The goal: **get them a real, deployed win within 48 hours of joining** — not "learn about AI," but actually produce something they use on a real listing or real client THIS WEEK. Blends Saraev's accountability-first daily structure, Herk's polished tutorial delivery, and Hormozi's speed-to-value obsession.

**Hormozi's rule: The first 48 hours determine if someone stays or churns.** Every element of this module is designed around that.

**Primary tool: Claude Cowork (Claude Code).** Every video in this module is recorded using Claude Cowork. Mention this at the start of every video when you first show your screen. Members who use regular Claude (browser) can follow along — it just adds a few extra steps (copy-pasting prompts vs. using skills/commands). The setup videos at the beginning cover both paths.

**Design philosophy:**
- Daily tasks, not passive courses — each day has ONE action with a real, tangible output
- Each lesson is completable in 15-30 minutes
- Every lesson ends with "Post your result in Wins & Results" (public commitment = retention)
- Screen-share walkthroughs with light editing (Herk polish) — cut dead air, add text callouts, clean audio. NOT raw Looms, NOT studio production. Professional enough to feel like a real course, authentic enough to feel like a working agent showing you his screen
- **Hormozi's production hierarchy:** Script quality > audio quality > lighting > camera. Invest in a good mic before anything else.
- **Show YOUR real listings and real workflows** — you're a working agent, prove it in every video.

### Module 0 Unlock Logic

The module has **9 total lessons**: 2 setup videos + 7 daily workflow lessons.

**Immediately unlocked on join (no gate):**
- Setup A: Claude Cowork Tutorial
- Setup B: Claude Project Setup (for non-Cowork users)
- Day 1: Your First Listing Description in 90 Seconds

**Locked until Day 1 is marked "Complete":**
- Days 2 through 7 (visible but locked — members can see titles and descriptions)

Once Day 1 is marked complete, **all remaining lessons unlock at once.**

The two setup videos are optional — members choose whichever path fits them (Cowork or browser Claude) and can skip straight to Day 1 if they're already set up. No gate on the setup videos. No gate between Setup A and Setup B. The only gate in the entire module is Day 1 unlocking Days 2-7.

**Why this structure works:**
- Members can see what's coming — builds anticipation and shows the full value of the module
- The setup videos remove the "I don't know how to get started with the tool" friction before Day 1
- The gate on Day 1 ensures they actually DO the quick win before moving on (action-first culture)
- Fast movers aren't artificially slowed down — if someone wants to crush all 9 in a weekend, let them
- Slower members still see the roadmap and know exactly where they're headed

**Skool setting:** Set Day 1 as the prerequisite for Days 2-7. Setup A and Setup B have no prerequisites.

You are a real estate listing copywriter specializing in MLS descriptions. Write a single, publication-ready MLS listing description for the property below.

"""  
[paste your previous listing description here]  
"""

- Address: [address]
- Beds/Baths: [X] bed / [X] bath
- Square footage: [X]
- Year built: [X]
- Key features: [list 5–8 features]
- Neighborhood/community: [name]

> **#1 task I want to automate:** [free text]
> **AI experience level:** [never used / tried a few times / use occasionally / use daily]

This creates public commitment (Hormozi: people who make public commitments are 3x more likely to follow through) and gives you data on what your members actually need.

On Day 2, post a thread (or have it auto-posted): "Day 2 check-in — who completed Day 1 and posted their listing description? Drop a comment if you're in." Tag new members who haven't posted. This gentle accountability is the difference between 20% completion and 60% completion.

### 4. Win Celebration System
Every time someone posts in Wins & Results, you (or a moderator) comment on it with genuine feedback. Hormozi: the community owner celebrating wins creates a culture where members WANT to post wins, which creates social proof, which sells the community for you. It's a flywheel.
