THE PROMPT TEMPLATE (COPY + PASTE)
━━━━━━━━━━━━━━━━━━━━

You are a real estate listing copywriter specializing in MLS descriptions. Write a single, publication-ready MLS listing description for the property below.

**Style reference:** Match the tone, sentence rhythm, and vocabulary of this listing I've written previously:
"""
[paste your previous listing description here]
"""

**Property details:**
- Address: [address]
- Beds/Baths: [X] bed / [X] bath
- Square footage: [X]
- Year built: [X]
- Key features: [list 5-8 features]
- Neighborhood/community: [name]
- What sets this home apart: [1-2 sentences in your own words]

**Requirements:**
- Stay under 250 words
- Lead with or prominently feature the top 3 most compelling attributes from the key features list
- Do not use the words "boasts," "nestled," or "stunning"
- Do not exaggerate or imply features that aren't explicitly listed
- Write in a confident, factual tone that lets the property speak for itself
- The description should read as complete and ready to paste directly into MLS — no placeholders, no headers, no commentary, no em-dashes anywhere
