## PART 5 — After Upload (still in Studio)

### A. Thumbnail A/B test
- Go to the video → **Test & compare** (right side)
- Upload 2 more thumbnail variants:
  - V2: split-screen "Before / After" agent
  - V3: listing description being generated with "90 SECONDS" overlay
- Let YouTube rotate automatically

### B. Verify chapters
- Watch the first few seconds of the published video — chapters should show on the progress bar. If not, make sure description has a `0:00` line and at least 3 chapters each ≥10 seconds.

### C. Set as channel trailer
- Back to **Customization → Layout → Video spotlight → Channel trailer** → select this video → Publish

After flipping to Public (next step), post this as a comment on the video and click the 3-dot menu → Pin:
