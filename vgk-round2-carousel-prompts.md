## Tips for Consistency
- Use the same seed number across all prompts (in Midjourney: `--seed 12345`) to maintain a consistent style
- If one image comes out significantly different, regenerate with `--sref [URL of your favorite result]` to style-match
- In Ideogram: use "Cinematic" style preset for all
- Always append `--ar 4:5` in Midjourney or set custom dimensions to 1080x1350
- If any image generates text/words, add `--no text, words, letters, numbers` to the prompt
